Synopsis

DataManInfo is a class to manipulate a datamanager info record and table descriptions. Often an existing table description and datamanager info record is used to construct a new table, but it might be necessary to change it somewhat.

  • Remove hypercolumn definitions from a table description. They are not needed anymore and can be a burden.
  • Replace non-writable storage managers (like LofarStMan) by a writable one.
  • Replace the deprecated TiledDataStMan by TiledShapeStMan.
