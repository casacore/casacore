Detailed Description

template<class T>
class casacore::GNoiseFunction< T >

A one dimensional normal distribution.

Intended use:

Public interface

Review Status

Test programs:
tFunctionHolder

Prerequisite

  • GNoiseParam
  • Function

Etymology

Gaussian Noise generator.

Synopsis

A GNoise is described by a mean and a variance (Note these are not parameters in the Function sense, but more like the order of a polynomial. The defaults are 0 and 1.

Example

GNoiseFunction<Double> sf;
sf(); // = 0.12

Template Type Argument Requirements (T)

  • T should have standard numerical operators

Thrown Exceptions

  • AipsError if incorrect parameter number specified.
