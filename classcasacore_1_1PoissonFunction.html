Detailed Description

template<class T>
class casacore::PoissonFunction< T >

A one dimensional Poisson function.

Intended use:

Public interface

Review Status

Test programs:
tFunctionHolder

Prerequisite

  • PoissonParam
  • Function

Etymology

A 1-dimensional Poisson.

Synopsis

A Poisson is described by lambda. The parameters are enumerated by LAMBDA. They have default values of 1.

Example

PoissonFunction<Double> sf(5.0);
sf(25); // = 5.0

Template Type Argument Requirements (T)

  • T should have standard numerical operators

Thrown Exceptions

  • AipsError if incorrect parameter number specified.
