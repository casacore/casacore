Detailed Description

Abstract base class for aggregate functions giving an integer scalar.

Intended use:

Internal

Review Status

Test programs:
tExprGroup

Synopsis

This class is derived from TableExprGroupFuncBase and acts as the abstract base class for aggregate functions resulting in an integer scalar.
Derived classes can use itsValue to contain the aggregated value. It that case they do not need to implement the get function.
