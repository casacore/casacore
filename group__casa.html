Detailed Description

The casa package contains the core modules.

  • Multi-dimensional arrays
  • Quantities (values with physical units)
  • OS and IO interface classes
  • Containers (such as Record)
  • Various utility classes and functions
  • Classes to store and retrieve Casacore data types in/from the HDF5 file format
  • Classes to store and retrieve Casacore data types in/from the Json file format
  • Miscellaneous modules for exceptions, tasking interface, command line input, logging, STL wrappers, and basic math classes and functors

It is important to note that most of the code has been developed before STL came into existence. Several classes (such as Map) in module Containers are superseded by their STL counterparts and are not used anymore in the Casacore code. However, for backward compatibility the classes still exist, but are not built unless a special Cmake option is given.

Note that class Regex still exists because std::regex is not supported in gcc-4.8 and before; this compiler is still shipped in some distributions. Furthermore, some classes (such as String) offer quite some extra functionality compared to STL.
