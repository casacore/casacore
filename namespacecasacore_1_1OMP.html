◆ maxThreads()

uInt casacore::OMP::maxThreads ( )
inline

Get the maximum number of threads.

OpenMP sets it to the env.var. OMP_NUM_THREADS. If undefined, it is the number of cores. If OpenMP is not used, 1 is returned.
