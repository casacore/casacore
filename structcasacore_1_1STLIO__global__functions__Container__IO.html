Detailed Description

Input/output operators for STL-like containers.

Intended use:

Public interface

Review Status

Reviewed By:
Paul Shannon
Date Reviewed:
1995/02/21

Prerequisite

  • STL container concept

Synopsis

The function showContainer makes it possible to show any STL-like container (having forward iterators) on an ostream. This include casacore classes like Array, IPosition, and Block, but also STL classes like vector. The separator, prefix, and postfix can be defined at will (they default to, [ ]).

The function showDataIter is similar to showContainer, but uses iterators directly.

Example

IPosition shape (3,10,10,3);
showContainer (cout, shape);

Motivation

Effortless input/output is clearly a big win.
